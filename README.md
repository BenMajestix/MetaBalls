# MetaBalls
2D Implementation of Metaballs (https://en.wikipedia.org/wiki/Metaballs) with Controls in C++ &amp; SDL2. 

### Controls:
Escape -> Quit
F1 -> Show Balls
F2 -> Hide Balls
1 -> Slow Motions / Reverse Motion
2 -> Advance Motion
F3 -> Show Steps / Reduce Resolution
F4 -> Remove Steps / Higher Resolution

### Libraries to Build
- SDL2
- SDL2 GFX Primitives

### Build
make build
